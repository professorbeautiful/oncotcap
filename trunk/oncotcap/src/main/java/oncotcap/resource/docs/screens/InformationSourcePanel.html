Information Source editor window


PURPOSE:
  To record an information source from which knowledge nuggets are to be extracted.

HOW TO USE THIS WINDOW:
 

Fill in the fields for the article (but see below for a shortcut).
To add a Knowledge Nugget, click the "add" button to the Knowledge Nugget list.

TIPS:
 

Use your favorite method to find an article's PubMed ID (PMID) and place it in the clipboard.
For example, search with PubMed, then mouse-select the PMID and type control-C ("copy").

Finally, click the "get" button. (There's no need to paste.)
The other fields will now fill automatically.


KNOWN BUGS & INCONVENIENCES:
 

The author list is alphabetical. This will be fixed in subsequent versions.

The full-text URL finder may not work for some journals, depending on what full-text providers your institution subscribes to.
