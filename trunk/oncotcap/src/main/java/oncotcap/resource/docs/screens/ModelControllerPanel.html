ModelController editor window


PURPOSE:
  To package one or more SubModelGroups and configure them for use in a modeling application.

HOW TO USE THIS WINDOW:
 

Create a new ModelController from the browser tree by clicking on in the right-hand toolbar whenever it is visible. If a SubModelGroup object is selected in the tree, the new ModelController will be linked to it.

To configure the ModelController for use in an application, take the following steps:

  1. Set the Starter Object, e.g. Patient, Clinical Trial, Cell, Oncologist, etc.
  2. Add the desired SubModelGroup in the table
