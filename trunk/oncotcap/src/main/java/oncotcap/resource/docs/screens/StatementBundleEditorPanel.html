Statement Bundle editor window


PURPOSE:
  To represent an Edict as a Statement Template with values assigned to the parameters.

HOW TO USE THIS WINDOW:
 

You can obtain this window in several ways:

  • From an Edict editor window, click .
  • When an Edict is selected in the browser tree, click . The browser's tri-flow controller must have the leaf turned on.
  • When a StatementTemplate is selected in the browser tree, click . The browser's tri-flow controller must have the leaf turned on.
Except for the third way, a StatementTemplate must be selected, or created. The StatementBundle will be based on its StatementTemplate. Click on a parameter (in blue, with the appearance of a web page link). An editor window will appear for that parameter, in which values can be set.

TIPS:
  It is not necessary to assign values to all parameters. Unassigned values can be set later when the model is run. See ModelController.
