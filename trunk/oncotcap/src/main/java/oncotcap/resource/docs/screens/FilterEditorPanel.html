Filter panel


PURPOSE:
   To limit the entries in the Browser Tree to a selected subset.

HOW TO USE THIS WINDOW:
 

 Build a filter from the following elements:

  • A particular browser node object. This can be dragged in from the Browser tree on the right. Only objects connected to the given object are shown.
  • A browser node type. This can be dragged in from the Tricorn controller above, or from the root of the Browser tree.
  • A text string that a displayed . Click.This signifies a "contains string" relationship.
  • A Boolean (logical) operator, , , .

TIPS:
 

 Delete a node in the filter tree using the "delete" key.

Turn the filter on or off using the buttons in the toolbar to the right of the Browser Tree.
