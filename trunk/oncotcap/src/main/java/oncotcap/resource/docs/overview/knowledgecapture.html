Overview of the Knowledge Capture work context

 

Knowledge acquisition is essential for modeling, both for model-building and model-validation. The knowledge acquisition process should be well documented, so that others can retrace steps, review the judgments that were made , critique, revise and extend the work that contributed to the modeling.
