The tri-flow controller

 

The controller on the upper left sets the root node and leaf nodes of the browser tree on the right.

To set the root node of the tree, click on one of the wide icons labeled with a words. The white arrow moves to the selected icon.

To toggle a leaf node on or off, click on the small square icon to the right of the wide icon. When the small square icon is blank, the corresponding type of browsable object will not appear in the browser tree.
