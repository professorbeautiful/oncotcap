Chromosomal evolution model

This page briefly sketches some ideas for modeling the evolution of a tumor in fine detail,
as it would (will) be approached in the OncoTCap environment..

These are the key elements:

  • An edict that uses a gene allele expression level in a cell.
  • Gene expression is amalgamated from the states of each copy of the gene.
  • A collection of chromosome objects instantiated in the cell.
  • Each chromosome has a collection of genes.
  • Each gene has a state describing general accessibility to all TFs.
  • Each gene has a set of regulatory regions.
  • Each regulatory region has edicts to control its response to a TF.
  • Kinetics events that govern
    • point mutations
    • tetraploidy
    • chromosome mis-segregation (loss and gain)
    • gene amplification HSP
    • gene amplification DM
    • translocation
