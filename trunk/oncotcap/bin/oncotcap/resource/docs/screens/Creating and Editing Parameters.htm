Creating and Editing Parameters

In a Statement Template, to insert a parameter into the text, type alt-A or click the word

add addadd (upper right).

To delete the parameter, type backspace or delete when the text cursor is positioned next to the parameter.

To edit the parameter (change its name, its default value or any other configuration value), select it in the Statement Template's list of parameters (right side), and click Edit. An appropriate parameter editor window will appear.
