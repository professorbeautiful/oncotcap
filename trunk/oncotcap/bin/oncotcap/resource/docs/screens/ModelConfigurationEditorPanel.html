Model configuration editor panel


PURPOSE:
   Organize the ModelController's constituent StatementBundles into a tree for display.
Decide which will be visible to the end user of the particular modeling application.

HOW TO USE THIS WINDOW:
 

 Drag SB's from left to the right side, to make them invisible to the end user.

Right click on any left-side entry to add a label node.

Press the "delete" key to move a selected SB to the right (invisible) side.
