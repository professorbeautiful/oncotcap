Edict editor window


PURPOSE:
  To record your edict/rule/instruction/law/imperative for the model world which your interpretation is to be implemented.

HOW TO USE THIS WINDOW:
  Write your own Edict into the SentenceText text area.

TIPS:
 

Often a good way to start is to paste in the text from an Interpretation, then edit it into an imperative sentence structure. One can also paste in the text of a StatementBundle. (Parameters will be converted to ordinary text.) Or, just write it from scratch.

The purpose of the Edict is to represent a modeling idea with enough clarity. This means that it should be easy to judge whether an implementation (StatementBundle) faithfully represents the idea.

The Edict can be left blank, but that's probably asking for trouble.
