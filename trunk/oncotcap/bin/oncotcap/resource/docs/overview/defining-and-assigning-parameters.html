Defining and Assigning Parameters

When creating or editing a Statement Template, one adds parameters by clicking Add above the upper-right-hand panel. This brings up a pick-list for the type of parameter desired. Once this is chosen, the particular editor window appropriate fo rthat parameter type appears. In the top portion, you assign a name to the parameter. (In case of compound parameters, there is more than one name, one for each constituent parameter.)

x1This is the name to be used for backquoting, the mechanism for passing parameter values down to the Code Bundles and Statement Bundles that support this Statement Template.
