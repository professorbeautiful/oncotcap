Level lists

A level list is a finite list of allowed values for a "Characteristic" parameter.
(A Characteristic parameter could also be named "denumerable", "enumeration", or "enum", terms used in other systems and languages for the same idea.)

A level list is attached to a Keyword in the Keyword tree. We'll call that Keyword the "home node" for the particular level list. This level list is then available for use in any Characteristic parameter whose name is taken from the home node or from any other node "downstream" (a sub-node, sub-sub-node, etc) of the home node. Conversely, if you want to create a Characteristic and choose a keyword as its name, then that Characteristic can use any Level List which is attached to any Keyword upstream (parent node, grandparent node, etc).

The Level List editor is found by right-clicking on a Keyword in the Browser or in a number of parameter editors where Keyword trees appear.

Level List values are all converted into capital letters.
