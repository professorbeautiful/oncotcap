Treatment Schedule Parameters


PURPOSE:
   This parameter editor specifies a treatment schedule for administering treatments on a fixed set of days within a course, and repeating the course a fixed number of times,

HOW TO USE THIS WINDOW:
 

A treatment-schedule parameter is a compound parameter consisting of three individual parameters:

  • Day list , a string of numbers indicating the days of treatment administration within each course.
  • Duration, an integer number of days indication the duration of one course.
  • Courses, an integer number of courses to administer.

A click on any one of these individual parameters will launch an "editor" window for inputting or changing any of the values for the individual parameters..

TIPS:
   One can pass the information in this parameter into a Code Bundle, specifically into an Action for scheduling an Event, using the "backquote" mechanism. See Inheritance

KNOWN BUGS & INCONVENIENCES:
 

 This parameter can be used also for scheduling medical tests and other schedulable items, but then its name (and the word Courses) should be changed to reflect the more general use.
