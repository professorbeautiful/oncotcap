Overview of the Instruction-Mapping work context

Edicts are mapped to "instructions" using objects from the the lower left leg of the tri-flow controller. These instructions are agent-based descriptions that can in principle have many ways to be implemented in actual working code, depending on the computational "engine" to be used. (Currently, only the native agent-based simulator is support. Other agent-based and non-agent based simulators, including analytical solutions, can in principle be used instead.)
