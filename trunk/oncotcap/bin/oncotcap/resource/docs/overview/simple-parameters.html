Simple Parameters

See also Creating and Editing Parameters

Float, Integer, String, Positive, and Switchable are all simple parameters, or "single parameters".

When added to a Statement Template, a simple parameter will appear as a single word with a hyperlink appearance. In the Statement Template it is not clickable- the cursor will bounce off it, but when a Statement Bundle uses this Statement Template (by bundling in actual values for the parameters), it is clickable! A click will launch an "editor" window for inputting the value.
